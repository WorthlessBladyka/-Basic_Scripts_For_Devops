## The present Indefinite Tense (настоящее неопределенное время)

Он употребляется для обозначения обычного действия или состояния предмета, а также повторяющихся действий.

|                     |                    |
| ------------------- | ------------------ |
| They study well.    | Они учатся хорошо. |
| We live in Moscow.  | Мы живем в Москве. |
| You go to the club. | Вы ходите в клуб.  |

Спряжение глагола:

| ЛИЦО   | Ед. ч.                         | Мн. ч.                 |
| ------ | ------------------------------ | ---------------------- |
| 1 лицо | I take<br>я беру               | We take<br>мы берем    |
| 2 лицо | -                              | you take<br>вы берете  |
| 3 лицо | he(she, it)  takes<br>он берет | they take<br>они берут |
