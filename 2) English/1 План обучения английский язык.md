# Месяц 1:

1) Dualingo - изучение и повторение новых слов (10-20 минут)
2) Словарный запас (Vocabulary) "В течении дня"
	**Quizlet:** Позволяет создавать свои наборы карточек. Идеально для заучивания IT-терминов.
	**Anki:** Мощная система интервального повторения. Карточки.
	**Memrise:** Есть готовые курсы, в том числе и по базовому английскому, с видео носителей языка.
3) Английский по плейлистам A0 (30-40 минут) 
4) Самоучитель по английскому(грамматика) (30-60 минут)
5) Аудирование (5-30 минут)
	   **BBC Learning English - 6 Minute English:** Короткие подкасты на разные темы с текстовой расшифровкой и глоссарием.
	   **Easy English - YouTube Channel:** Короткие уличные интервью с субтитрами.
	   **EnglishClass101 - YouTube Channel:** Диалоги для начинающих с разбором.
6) Говорение (20-30 минут)
		**Speak & Improve (от Cambridge English):** AI-бот, который оценивает вашу речь.
		**ELSA Speak:** Помогает отработать произношение.
		**HelloTalk / Tandem:** Приложения для языкового обмена. Вы можете писать и говорить с носителями языка. Можно указать в профиле, что вы IT-специалист, и найти коллег для общения.
