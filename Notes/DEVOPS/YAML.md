# JSON and XML

- JSON (JavaScript Object Notation) - текстовый формат обмена данных, основанных на JS

Типы данных:
- строка
- число
- bool
- object
- array
- null
Не передается:
- function
- date
- undefined - не проиницилизированная переменная (иницилизируется, либо NULL, либо ...)

### Почему JSON лучше чем XML
- JSON лучше адаптирован к JS. В XML необходимо дополнительно утанавливать библиотеки
- Парсить XML вручную. Парсинг JS по умолчанию  встроен в JSON  
- JSON код более легковесный и более читабельный
- JSON есть массивы
- Неиспользует тэги
