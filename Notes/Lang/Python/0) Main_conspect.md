##### **Резка списков**
`list2 = list1[:]` - копируем список
`del list1[:]` - очистить список
```python
		# Если начальное значение резки указывает на конец списка с индексом -1, а конечное значение < len(list), разрезание вернёт пустой список. Если мы не укажем конечный параметр или конечный параметр будет `>= len(list)`, то нарезка вернёт последний элемент.
my_list = [1, 3, 7, 9] 
print(my_list[-1:-1]) 
print(my_list[-1:1]) 
print(my_list[-1:4]) 
print(my_list[-1:])

[] 
[] 
[9] 
[9]
```

В нотации нарезки также есть параметр шага: **`list[начало:конец:шаг].`**
Если мы укажем **отрицательный** индекс в параметре шага, нарезка будет **обратной**.
##### **Поиск в списках**
``` python
		# есть ли
list = [1,2,3,4,5]
print(1 in list)
---
True
```

``` python
		# отсутствует ли
list = [1,2,3,4,5]
print(6 not in list)
False
```

## Исключения и error
> Когда python пытается выполнить код и при его выполнение возникает ошибка, python останавленивает исполнение программы и создает особый тип данных =="исключения"==.
>  Если возникло исключение, оно ожидает, что кто-нибудь позаботится о нем
>  Однако, если ни кто о нем не позаботится программа будет завершена
>  Можно обработать исключения с помощью ключевых слов "try" и "except"
